Managing Workspace

Workspace is the place where all SVG files and other data are stored. The workspace is created on the Webionix cloud server and is available anytime. Any changes in the SVG data is always stored in workspace. Hence, it is necessary that application is always connected to the cloud.

Create New SVG File

  • Enter an unique filename to create a SVG file.

NOTE:Avoid any blank space in the name

Open a SVG File

  • Select a file from the list in the MyWorkspace dialog.
  • Click Open to open the file

NOTE:MyWorkspace dialog shows blank if no file exist in the workspace

Close a SVG File

  • Click on the button to close the existing SVG file opened in the editor

Group

One or more objects can be put into an user defined group. One can position a group of objects at a desired location by dragging the selected group

  • Enter an unique group name to create a new group.

Selecting a Group

  • Click on the Group item on the tree node under the Groups tab
  • All objects within the group are selected within a bounding rectangle

Deleting a Group

  • Select the Group.
  • Use the Delete button in the toolbar to delete the group

Adding Object to a Group

  • Select an Object in the Editor
  • Right Click on the mouse to invoke the context menu
  • Select the desired Group where you want to add the Object to

Adding Objects

Objects are nothing but various types of SVG objects. In Webionix we mainly deal with 4 different classes of objects

  • Shapes:Circle,Ellipses,Rectangles are examples of Shape type objects.
  • Path:Lines,Curves,Polygons, Freehand are examples of Path type objects.
  • Text:Text Objects are those which contain only textual matter.
  • Image:Image Objects are the Image objects containing URL to an Image

Adding Shape or Path Objects

  • Select the Group in which the Object is to be added.(By default an object is always added to BASEGROUP )
  • Select an Object from the list of Objects(except the Polygon)
  • A rectangular dot is visible which is a drawing pointer, which moves with the mouse cursor
  • Click at the point from where you want to start drawing
  • Drag the cursor downwards and towards right
  • Release the mouse button to mark the end of drawing

Adding Polygon Objects

Modifying Objects

Objects can be modified in various ways using the editor. In case of shape type objects only the dimension can be changed whereas in case of path objects the points can be modified.

Modifying Shape Type Objects

  • Select a Shape type object (e.g. Rectangle,Circle, Ellipse).
  • Drag the outline of the object to change the dimensions of the object.

Modifying Path Type Object

  • Select a Path type object
    (e.g. Lines,Cubic Bezier, Quadatric Bezier, Elliptic, Polygons etc)
  • Go to Properties tab and click on . One could see the points of the path marked with circles.
    The Green circle indicates Start point and the Red circle indicates End point of the path
  • Select the point you want to move and drag it to the desired location to modify the path

Modifying Image Object

  • Select the Image object
  • Drag the outline of the object to change the dimensions of the object.

Modifying Polygon Objects

  • Select the Polygon object
  • Click on the button,the editor shows all the points of the polygon.

Selecting Objects

To modify any property of an object, one needs to select an object. The editor allows one to select object in two modes

  • Single Object Selection: In this mode, just click on the target object and an Orange outline should appear
    OR
    One can also select an object from the Groups tab by selecting the appropriate tree item.
  • Multiple Object Selection: In this mode two or more objects can be selected at a time. There are two methods to achieve this.

      Method-1
    • Press "Ctrl" key and then select objects one by one. The first object selected will be marked in Red and the remaining ones will be in Green
      Method-2
    • On clicking the button in the toolbar, one can see rectanglar dot appearing on the editor
    • Start from some point in the editor and drag it downwards in such a way that all desired objects are covered within the dragged path and then release the mouse button
