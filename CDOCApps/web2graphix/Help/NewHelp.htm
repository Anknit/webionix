Managing Workspace

Workspace is the place where all SVG files and other data are stored. The workspace is created on the Webionix cloud server and is available anytime. Any changes in the SVG data is always stored in workspace. Hence, it is necessary that application is always connected to the cloud.

Create New SVG File

  • Enter an unique filename to create a SVG file.

NOTE:Avoid any blank space in the name

Open a SVG File

  • Select a file from the list in the MyWorkspace dialog.
  • Click Open to open the file

NOTE:MyWorkspace dialog shows blank if no file exist in the workspace

Close a SVG File

  • Click on the button to close the existing SVG file opened in the editor

Adding Objects

Objects are nothing but various types of SVG objects. In Webionix we mainly deal with 4 different classes of objects

  • Shapes:Circle,Ellipses,Rectangles are examples of Shape type objects.
  • Path:Lines,Curves,Polygons, Freehand are examples of Path type objects.
  • Text:Text Objects are those which contain only textual matter.
  • Image:Image Objects are the Image objects containing URL to an Image

Animation

The application supports SVG animations as per the SVG specification. One can animate objects created in the editor. You can also have multiple animations defined for a single object. The animation of any object is controlled by 3 basic parameters:

  • Animation Type: There are different types of animation supported which can be viewed in the list while adding any new animation.
  • Start Time of the animation: By default all animations are executed when SVG is loaded. Hence, animations by default starts at 0th second after loading. However, one may also synchronize the start time with the end or beginning of animation of some other object.
  • Duration:Duration defines the duration for which the animation will be executed. By modifying this parameter one can make an animation slow or fast.

NOTE:Any change in animation parameters or position of an object should be followed by click on the button to make the changes effective.

Adding Animation

  • Select an object which you to animate.
  • On Clicking the button, a Pop-up dialog appears as shown below
  • Enter an unique title for the animation.
  • Select an Animation type from the list.
  • Click on the Preview button to preview the animation.
  • On clicking OK button the Animation is added in the Animation list in the Animation tab.

Deleting Animation

  • Select the animation from the animation list.
  • On clicking button the animation is removed from the list.

To play an animation select an animation title and click on the button to preview the animation

Timing Controls of the Animation

The animations can be controlled from the Animation list and changing the Start parameters and Duration. By default all animation starts at 0th second.

NOTE:In case of multiple animations there should be at least one animation which starts at 0th second.

Start of Animation

  • Select an animation item from the animation from the list.
  • Click on the cell corresponding to “Start” column as shown below:

    .
  • Select one of the options below:
    At 0th second: In this case the animation will start as soon as the SVG is loaded.
    After:When this is is chosen,the animation starts right after the reference animation finishes.
    To choose a reference animation click on the cell corresponding to Reference column and select one of the animations as reference.
    With:In this case, the animation starts in sync with begining of the reference animation.
    To choose a reference animation click on the cell corresponding to Reference column and select one of the animations as reference.

Duration of Animation

Duration of an animation defines the time for which the object will be animating. Shorter duration will give an effect of fast animation and longer duration will give an effect of slow animation.

  • To modify duration just click on the cell corresponding to the Duration column in Animation List.
  • Enter the time (in seconds) and press the Apply button

Repeatition of Animation

One can make an animation repeat mutliple times by simply modifying the value in the input-box

Paramters Specific to Animation Types

There are parameters which are specific to certain types of animation.

Rotation

In the Rotation type animation, an object is Rotated from an Initial angle to a given Final angle around a given Center of rotation.

Start Angle

Gives the initial angle from where the object should start rotating

  • Enter the value in to define the Start angle.
  • Click on the Apply button

End Angle

Gives the final angle to which the object will rotate to.

  • Enter the value in to define the Start angle.
  • Click on the Apply button

Auto Reverse

Switching On this flag, will allow the rotation to swing back to initial position

Center of Rotation

The center of rotation is point around which the rotation takes place. One can move the center of rotation.

  • Take the mouse on the grey dot which is the center of rotation.
  • Drag the dot to a desired location.
  • Click on the Apply button to apply the new position of the Center of Rotation.

Path Aligned Motion

In the path aligned motion an object is moved along a given path. The Start point of the path is the Starting point and end point of the path being the end of animation. By default the initial path is a straight line which can be later modified into any other path of choice.
The following parameters are specific to Path Aligned Motion.
